Project Hunter

Menus

Project Hunter is an ARPG game that i have been developing that combines rouge like gameplay like diablo and Path of exile with a Souls like game play. With fully integrated equipment menu allowing the player to equip and remove items.

Interaction System

The interaction system, comprised of the Interaction Manager and Interactable Manager, is a dynamic, modular component of an Unreal Engine 4 game project designed to facilitate player interactions with the game environment and NPCs. The Interaction Manager tracks and prioritizes interactive objects within the player's vicinity, updating in real-time. On the other hand, the Interactable Manager is attached to each interactive object, defining the nature and response of interactions. Together, they provide a versatile tool for creating rich, interactive game environments, driving immersion and intuitiveness in gameplay.

Item Management

Develop a fully customizable weapon system that allows coders to effortlessly incorporate new weapons into the game by simply defining the item's type, mesh, image, and other relevant properties. This versatile solution enhances the efficiency and flexibility of game development while promoting ease of maintenance and scalability.

Randomized Loot

To fully embody the aspect of an Action Role-Playing Game (ARPG) in this game, the loot system generates items randomly using randomly generated numbers. The first number determines the highest rarity level of the item, while subsequent rolls determine the quantity of drops for each item. These settings are fully customizable, and even allow for manually specifying which items should be guaranteed to drop
